# Salon1
This is a website of a beauty salon that allows customers to check the offer of this salon. On the admin page you can change information on the website such as prices, add photos, add blog pages. You can also approve comments added to the blog page. As a user, you can contact the salon via the form on the website to arrange an appointment, after which you will receive an automatically generated message.
